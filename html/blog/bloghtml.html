1. Introducción a HTML


HTML es el acrónimo de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), y se utiliza para crear y diseñar páginas web. Es un lenguaje de marcado que permite la creación de estructuras de contenido en línea y es la base fundamental de cualquier página web.

En este texto, vamos a explorar algunos de los conceptos fundamentales de HTML y explicar cómo se utiliza para crear páginas web.

Para empezar, es importante comprender que HTML es un lenguaje de marcado de texto plano. Esto significa que todas las etiquetas de HTML están escritas en texto plano, sin gráficos ni otros elementos visuales.

Una etiqueta de HTML se compone de dos partes principales: el elemento y su contenido. El elemento es el nombre de la etiqueta, como por ejemplo "p" para párrafo o "h1" para el encabezado de nivel 1. El contenido es lo que se encuentra dentro de la etiqueta, como texto, imágenes, enlaces y otros elementos.

Las etiquetas de HTML se escriben entre corchetes angulares ("<" y ">"). Por ejemplo, la etiqueta de párrafo se escribiría de la siguiente manera:

<p>Este es un párrafo de ejemplo<p>

En este ejemplo, "p" es el elemento y "Este es un párrafo de ejemplo" es el contenido.

Ahora que hemos visto cómo se escriben las etiquetas de HTML, es importante entender cómo se utiliza para crear una página web completa. Una página web típica se compone de varias secciones diferentes, incluyendo la cabecera, el cuerpo y el pie de página.

La cabecera de una página web suele contener información importante, como el título de la página, la descripción y las palabras clave. Estos elementos se utilizan para mejorar la optimización de motores de búsqueda (SEO) y ayudar a los usuarios a encontrar su sitio web en línea.

El cuerpo de la página es donde se coloca todo el contenido, como texto, imágenes y enlaces. También se pueden utilizar elementos como tablas, listas y formularios para organizar el contenido y mejorar la usabilidad.

El pie de página es donde se colocan elementos como el copyright, la información de contacto y otros elementos que no son necesarios para el contenido principal de la página.

En HTML, se utilizan etiquetas específicas para cada una de estas secciones. La etiqueta "<head>" se utiliza para la cabecera de la página, la etiqueta "<body>" se utiliza para el cuerpo y la etiqueta "<footer>" se utiliza para el pie de página.

Por ejemplo, aquí está el código HTML para una página web simple:

<!DOCTYPE html>
<html>
<head>
<meta name="description" content="Descripción de la página">
<meta name="keywords" content="Palabras clave de la página">
</head >
<body >
<h1>Encabezado de nivel 1 </h1>
<p>Este es un párrafo de ejemplo. </p>
<img src="imagen.jpg" alt="Descripción de la imagen">
<a href="https://www.ejemplo.com">Enlace de ejemplo</a>
</body>
<footer>
<p>Derechos de autor © 2023 Ejemplo</p>
<p>Contáctanos en info@ejemplo.com</p>
</footer>
</html>

En este ejemplo, la etiqueta <head> contiene el título de la página, la descripción y las palabras clave.

La etiqueta <body> contiene todo el contenido principal de la página, como el encabezado de nivel 1, el párrafo de ejemplo, la imagen y el enlace. Y la etiqueta <footer> contiene información sobre los derechos de autor y cómo contactar con el propietario del sitio web.

Además de las etiquetas de sección, existen muchas otras etiquetas de HTML que se pueden utilizar para agregar elementos específicos a una página web. Algunas de las etiquetas más comunes incluyen:

  • <h1> a <h6>: estas etiquetas se utilizan para crear encabezados de diferentes niveles. <h1> es el encabezado de nivel más alto y <h6> es el encabezado de nivel más bajo.
  • <p>: esta etiqueta se utiliza para crear párrafos de texto.
  • <a>: esta etiqueta se utiliza para crear enlaces a otras páginas web.
  • <img>: esta etiqueta se utiliza para insertar imágenes en una página web.
  • <ul> y <li>: estas etiquetas se utilizan para crear listas con viñetas.
  • <ol> y <li>: estas etiquetas se utilizan para crear listas numeradas.
  • <table>, <tr>, <th> y <td>: estas etiquetas se utilizan para crear tablas.
  • <form>, <input> y <button>: estas etiquetas se utilizan para crear formularios en línea.

Al escribir HTML, es importante tener en cuenta la estructura y la jerarquía de la página. Las etiquetas deben anidarse correctamente y los elementos deben estar colocados en el lugar correcto dentro de la estructura de la página. Esto ayuda a asegurar que la página sea legible y fácil de entender para los usuarios.

Además de HTML, hay otros lenguajes de marcado que se utilizan junto con HTML para crear páginas web. Uno de los más comunes es CSS (Cascading Style Sheets), que se utiliza para diseñar y dar estilo a una página web. JavaScript es otro lenguaje que se utiliza para agregar interactividad y dinamismo a una página web.

En conclusión, HTML es el lenguaje fundamental para la creación de páginas web. Con una comprensión básica de las etiquetas de HTML y la estructura de una página web, puedes crear páginas web simples y efectivas. Si quieres llevar tus habilidades de desarrollo web al siguiente nivel, es importante aprender también CSS y JavaScript. Con estas herramientas, puedes crear páginas web dinámicas y con estilo que cautiven a los usuarios y los mantengan interesados en tu sitio web.
