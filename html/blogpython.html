1. Introduccion



¿Que es python?

Python es un lenguaje de programación popular. Este fue creado por Guido van Rossum y lanzado en 1991. Funciona en diferentes plataformas como: Windows, Mac, Linux, Raspberry, Pi, etc. La sintaxis de Python permite a nosotros los desarrolladores escribir programas con menos líneas en comparación con otros lenguajes a la hora de programar.



¿Qué puede hacer Python?

Python es un lenguaje de programación versátil capaz de adaptarse a múltiples usos, algunos ejemplos de sus usos más comunes son:

  • Desarrollo web (servidor)
  • Creación de aplicaciones web
  • Manejo de archivos
  • Manejo Base de datos (MySql/MongoDB)
  • Ciencia de datos
  • Inteligencia artificial (AI)

2. Instalación de Python



Primero comprobaremos que no tengamos instalado Python en nuestro PC. En Windows ejecutaremos el siguiente comando en la aplicación “Símbolo del sistema” (cmd.exe):

< C:\Usuario\Tu nombre>C:\Usuario\Tu nombre>py –version >

Para comprobar en Linux o Mac, abrimos la terminal y escribiremos:

< C:\Usuario\Tu nombre>C:\Usuario\Tu nombre>python –version >

3. Inicio rápido

Python es un lenguaje de programación interpretado, esto significa que, nosotros como desarrolladores, escribiremos archivos de Python (.py) en un editor de texto y luego introducimos los archivos en el intérprete de Python para que se ejecuten.


Para iniciar rapidamente, crearemos un nuevo archivo “holamundo.py” en nuestro editor de texto favorito, para continuar escribiendo las primeras líneas de código:

< C:\Usuario\Tu nombre>print(“¡Hola Mundo!”) >
*print: permite imprimir por pantalla la variable o las variables entre paréntesis.

Debemos guardar el archivo antes de hacer que sea interpretado. Abriremos la consola, debemos estar en la ruta correcta de la ubicación del archivo para que este ejecute bien, escribiremos en la consola de nuestro sistema windows:

< C:\Usuario\Tu nombre>py holamundo.py >

Consola Linux o Mac:

< C:\Usuario\Tu nombre>python holamundo.py >

4. Líneas de comando de Python

Para probar una pequeña cantidad de código en python, a veces es más rápido y más fácil no escribir el código en un archivo. Esto es posible porque Python se puede ejecutar como una línea de comando. Para esto en la la consola del sistema ejecutaremos la siguiente línea de comando:

< C:\Usuario\Tu nombre>python >

Si el comando "python" no funcionó, probaremos con "py":

< C:\Usuario\Tu nombre>py >

Desde acá, podemos escribir cualquier comando de python, incluido el ejemplo de “¡hola mundo!” que acabamos de aprender.
